# The Price of Liberty – Iran Interactive Map (Leaflet)

## What this is
A free, static site (no backend) with:
- dark map
- clickable points (popups with photobook)
- LEFT SIDEBAR controls (search + year range + counter)
- marker clustering
- Iran outline overlay

## Files
- index.html
- styles.css
- script.js
- people.geojson
- iran_boundary.geojson
- tulip_icon.svg
- Photos/ (images)
